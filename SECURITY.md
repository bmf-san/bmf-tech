# Security Policy

## サポート対象

このリポジトリは `main` ブランチの最新デプロイのみを対象とします。過去のデプロイは個別にパッチされません。

## 脆弱性の報告

セキュリティ上の問題を発見した場合は、**公開 Issue にはしないでください**。

以下のいずれかで非公開に報告してください:

- [GitHub Private Vulnerability Reporting](https://github.com/bmf-san/bmf-tech/security/advisories/new)

報告時には以下を含めてください:

- 影響範囲の説明
- 再現手順 / PoC
- 修正案（任意）

**7 日以内**に受領の連絡をします。修正は深刻度に応じて **30 日以内**を目安に対応します。
