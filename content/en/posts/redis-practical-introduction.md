[Practical Redis Introduction: From Technical Mechanisms to Field Applications](https://amzn.to/412eRyA) was a book I read.

As the title suggests, it contains practical content, but it also covers so much about Redis that you might want to add 'detailed explanation' to the title.

There are surprisingly few books written in Japanese about Redis, making this book a valuable resource.

What I particularly liked was the thorough coverage of operational management. Although my understanding is shallow, the sections on internal implementation were also beneficial.

I want to keep this book handy for reference whenever I encounter issues with Redis.
