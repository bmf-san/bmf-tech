# Overview
A note on the classification of representative AWS services.

# AZ Services
- Subnet (does not span multiple AZs)
- NAT
- Network ACL
- EC2 Instance
- RDS Instance
- Elastic Cache
- Elastic File System
- Elastic Load Balancing
- EBS Volume
- Redshift

# Region Services
- VPC (can span multiple AZs)
- Security Group
- VPC Endpoints
- VPC Peering
- Elastic IP
- Auto Scaling
- S3
- Glacier
- Snapshot (EBS/RDS)
- AMI
- DynamoDB
- SQS
- SNS
- CloudSearch
- CloudWatch
- Lambda
- API Gateway

# Global Services
- IAM (Users, Groups, Policies, Roles)
- Route53
- CloudFront
- CloudTrail
- WAF
- STS
