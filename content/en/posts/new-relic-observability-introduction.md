[New Relic Practical Guide Second Edition: Fundamentals and Implementation of Observability](https://amzn.to/4cmAsFm) has been read.

It covers all the features of New Relic. Nearly half of the pages are dedicated to usage methods, allowing you to learn about practical applications.

I thought it would be nice if the specifications of NRQL were also included, but it seems quicker to supplement the missing parts with documentation or by actually trying it out.

- Telemetry data is defined as follows:
  - Metrics
    - A collection of measurements that are grouped or collected periodically
    - Data aggregation over a specific period
  - Logs
    - Text lines generated by the system when specific code is executed
  - Traces
    - The state of events or transactions that are interconnected
