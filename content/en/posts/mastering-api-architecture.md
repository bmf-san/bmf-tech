[Mastering API Architecture - Practical Techniques for Evolving Architecture from Monolithic to Microservices](https://amzn.to/3Uw15l6) is a book I read.

The introduction includes explanations of C4 diagrams and ADR, which I found beneficial. The discussions on architecture in this book are explained using these techniques.

It covers API architecture patterns, testing, operations, security, and more.

I believe it is almost impossible to avoid APIs on the web, so reading this book is worthwhile.

The term 'mastering' in the title gives the impression of being close to 'complete understanding.'

It is more of an overview rather than a detailed explanation, so for individual topics, it is necessary to supplement with reference books and materials.
