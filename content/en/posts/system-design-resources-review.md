I have been exploring various resources related to system design, so I will summarize them in a resource collection format. There are many videos that I haven't been able to watch yet. Additionally, some content may include topics that seem somewhat unrelated.

# Motivation
The reason I started researching is that I wanted to know if I could learn about system design in a systematic way and I wanted to learn more about it. To put it another way, I was looking for a way to enhance my skills in system design.

I believe that the ability in system design is largely dependent on craftsmanship and experience, but I thought there might be many things that are good to know as knowledge, so I tried to read a wide range of articles.

Of course, just reading them doesn't mean my skills have improved, and I don't feel that way at all, but I feel like I have learned quite a bit about what I should know and the points to consider, so I feel like I have gained useful information for the future.

# Resource List
## GitHub
- [donnemartin/system-design-primer](https://github.com/donnemartin/system-design-primer/blob/master/README-ja.md)
  - Covers the basic concepts of system design. It seems to be maintained, but there are some links to 404 articles and content that looks outdated.
- [shashank88/system_design](https://github.com/shashank88/system_design)
  - A collection of materials for starting to prepare for system design interviews.
- [yangshun/tech-interview-handbook](https://github.com/yangshun/tech-interview-handbook)
  - Content related to coding interviews rather than system design.
- [charlax/professional-programming](https://github.com/charlax/professional-programming) 
  - Not specifically focused on system design, but insights about software development in general. There are discussions about system design as well. It covers a wide range of areas necessary for engineering.
- [karanpratapsingh/system-design](https://github.com/karanpratapsingh/system-design)
  - Basic concepts of system design are succinctly summarized.
  - Good basic content that organizes a variety of topics.
- [checkcheckzz/system-design-interview](https://github.com/checkcheckzz/system-design-interview)
  - A collection of article links related to system design.
- [binhnguyennus/awesome-scalability](https://github.com/binhnguyennus/awesome-scalability)
  - A collection of article links related to system design. There are many resources categorized by keywords.
- [System Design Cheatsheet](https://gist.github.com/vasanthk/485d1c25737e8e72759f)
  - More like a list of keywords related to system design than a cheat sheet.
- [ByteByteGoHq/system-design-101](https://github.com/ByteByteGoHq/system-design-101)
  - Easy to understand with accompanying diagrams.
- [mehdihadeli/awesome-software-architecture](https://github.com/mehdihadeli/awesome-software-architecture)
  - Information aggregated about software architecture.

## Articles
- [Scalable Web Architecture and Distributed Systems](https://www.aosabook.org/en/distsys.html)
  - An example of best practices when building the web as a distributed system.
- [Enterprise Integration Patterns - Messaging Patterns](https://www.enterpriseintegrationpatterns.com/patterns/messaging/toc.html)
  - A collection of examples regarding messaging patterns.
- [tianpan.co](https://tianpan.co/notes/2016-02-13-crack-the-system-design-interview)
  - Briefly describes architecture examples and theories with minimal explanations.
- [hiredintech.com](https://www.hiredintech.com/system-design)
  - A step-by-step explanation of approaches to system design interviews, with example questions.
- [Technical Interview Questions](https://www.interviewbit.com/technical-interview-questions/)
  - A site for learning about technical interviews. It seems quite useful for job hunting in overseas tech companies.
  - Various quizzes are available, and there is a category for system design. There seem to be many questions and answers (discussions?), which could be educational.
- [How to Nail the System Design Interview + Top System Design Interview Questions and Answers](https://blog.tryexponent.com/how-to-nail-the-system-design-interview/#)
  - An overview of approaches to system design interviews.
- [System Design Interview Questions – Concepts You Should Know](https://www.freecodecamp.org/news/systems-design-for-interviews/)
  - A collection of tips on what you should know when tackling system design interviews.
- [highscalability.com](https://highscalability.com/)
  - Some articles from this site are also introduced in system-design-primer. Rich in case studies from various companies.
- [How to prepare for the System Design Interview in 2023](https://www.educative.io/blog/how-to-prepare-system-design-interview)
  - A well-organized article about preparing for system design interviews.
- [Google System Design Interview Preparation Doc | Complete Guide](https://workat.tech/system-design/article/google-system-design-interview-prep-doc-m684to5zzkj4)
  - A summary of preparation for Google’s system design interview.
- [An Introduction to System Design Interviews December 20, 2021](https://viglucci.io/introduction-to-system-design-interviews)
  - An introductory article about system design interviews. Briefly describes the approach.
- [System Design Interviews: A Step-By-Step Guide](https://designgurus.org/blog/step-by-step-guide) 
  - Describes the approach to system design interviews.
- [6 Interview Questions for System Designers (With Example Answers)](https://www.indeed.com/career-advice/interviewing/system-design-interview-questions)
  - Simple examples and answers for system design interviews.
- [Crack the System Design interview: tips from a Twitter software engineer](https://zhiachong.medium.com/how-to-system-design-dda63ed27e26)
  - Tips for system design interviews from a Twitter engineer.
- [System Design Interview Tutorial – The Beginner's Guide to System Design](https://www.freecodecamp.org/news/system-design-interview-practice-tutorial/)
  - An approach to interviews using system design examples on YouTube.
- [The complete guide to cracking the System Design interview](https://towardsdatascience.com/the-complete-guide-to-the-system-design-interview-ba118f48bdfc) 
  - Approach methods and a collection of example links for system design interviews.
- [System design interview prep (relax, start here)](https://igotanoffer.com/blogs/tech/system-design-interview-prep)
  - Basics about preparing for system design interviews.
- [System Design Interview Preparation Tips](https://www.interviewkickstart.com/blog/system-design-interview-preparation-tips)
  - Tips for system design interviews.
- [Architecture Issues Scaling Web Applications](http://venkateshcm.com/2014/05/Architecture-Issues-Scaling-Web-Applications/)
  - An article that succinctly states the key points about application scaling and performance tuning.
- [Little's Law in Performance Load Testing](https://qiita.com/hiro0107@github/items/6154b412c7ff29c8785f)
  - About Little's Law in performance load testing.
  - It seems good to remember this law when considering performance.
- [How To Determine Web Application Thread Pool Size](http://venkateshcm.com/2014/05/How-To-Determine-Web-Applications-Thread-Poll-Size/)　
  - How to determine the thread pool size for web applications.
  - An approach considering Little's Law.
- [principles.design](https://principles.design/examples/)
  - Collects design principles not limited to system design.

## Books
- [Fundamentals of Software Architecture](https://amzn.to/3y6m7hS)
  - A book discussing evaluation criteria and architecture patterns of software architecture.
- [Software Architecture Hardparts](https://amzn.to/44mx0Z6)
  - A book discussing difficult problems in approaches to distributed systems.
  - I want to read it again...
  - It’s a book I helped with as a volunteer for translation review, so I have a personal connection to it.
- [Data-Oriented Application Design](https://amzn.to/3UC7RGD)
  - It seems to stray quite a bit from the topic of system design, but I think there are many parts that relate to the design of distributed systems, so I included it.
- I read [System Design Interview](https://amzn.to/44oiVdI) since it was published.
    - The Japanese translation of [System Design Interview – An insider's guide](https://amzn.to/3N8srJE).
    - While I already knew some of the content from reading [donnemartin/system-design-primer](https://github.com/donnemartin/system-design-primer/blob/master/README-ja.md), it seems that not many books of this kind are published in Japan, so I want to keep a copy on hand.
    - [ByteByteGo](https://bytebytego.com/)
      - A service run by the author.
      - It seems to be mostly a paid service with some free content, but it might be worth subscribing as it seems educational.
      - There was a mention in the afterword about a Discord channel where the author discusses system design, so I joined that.
