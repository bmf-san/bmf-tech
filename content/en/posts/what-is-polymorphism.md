# Overview
Summary of polymorphism

# What is Polymorphism
- Poly (multiple) Morphism (shape)
- Providing a shared interface for multiple different types
- Instances receiving arguments behave differently depending on their class

# Types of Polymorphism
- Ad-hoc Polymorphism
  - Functions have different implementations for different types of arguments
  - Achieved through overloading and function overloading
- Parametric Polymorphism
  - Code is written without type specifications, allowing transparent use for each type
  - Can be applied to various types by passing types as arguments
  - Achieved through generics and templates
- Subtype Polymorphism
  - Handling multiple types with a common supertype under one name
