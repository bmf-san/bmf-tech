## Kenta Takeuchi

シニアプラットフォームエンジニア。

フリーランスのウェブ開発者としてキャリアをスタートし、スタートアップでの新規事業立ち上げ、ソフトウェアエージェンシーでの多様なプロジェクト経験を積む。その後、クラウドファンディング会社にソフトウェアエンジニアとして入社し、通知基盤や認証・認可プラットフォームなどの基盤システム開発に従事。現在は HR テック企業にてシニアプラットフォームエンジニアとして、プロダクトプラットフォームの開発・整備に取り組んでいる。

関わる組織とソフトウェアの長期的な価値を継続的に高めることを目標としており、システムアーキテクチャに特に関心がある。趣味は OSS 開発、筋トレ、観葉植物。

## 職歴

**SmartHR / シニアプラットフォームエンジニア** — 2024年6月〜現在

HR テック領域のプラットフォーム開発・整備、システムアーキテクチャ設計・再設計、パフォーマンス最適化。

**Makuake / シニアソフトウェアエンジニア・リードエンジニア** — 2018年12月〜2024年5月

プラットフォームチームのリードエンジニアとして、事業拡大に伴うスケーラビリティ課題を解決。認証・認可・通知基盤などの基盤システムの設計・開発をリード。

**Innovator Japan / プロダクトマネージャー・エンジニア** — 2017年5月〜2018年9月

求人関連サービスの新規サービス開発（企画・プロトタイプ）、CMS/EC パッケージ開発、受託 Web サイト開発。

**スタートアップ / Web エンジニア** — 2016年5月〜2017年3月

求人サービスの機能開発・UI 実装、アナリティクスを活用したサービス成長。

**フリーランス / Web デベロッパー** — 2015年6月〜2016年5月

多様なクライアント・プロジェクトにおける受託 Web 開発。
