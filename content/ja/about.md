## Kenta Takeuchi

シニアプラットフォームエンジニア。

フリーランスのWebエンジニアとしてキャリアをスタートし、スタートアップでの求人サービス開発、受託開発企業での多様なプロジェクト経験を積む。その後、クラウドファンディングのサービスを運営する事業会社にソフトウェアエンジニアとして入社し、通知基盤や認証・認可基盤などの基盤システムの設計・開発に従事。現在は人事・労務系SaaSを運営する事業会社にてシニアプラットフォームエンジニアとして、プロダクト基盤の開発に取り組んでいる。

組織・ソフトウェアの価値を長期的に向上させるアーキテクトを目指している。チームでスケールする技術の価値を重視しており、認証・通知・権限管理といった横断基盤の設計・開発経験から、一つの基盤が組織全体の開発効率や品質に与えるインパクトの大きさを実感している。技術戦略の策定と実行を通じて、事業成長を支える持続可能な技術組織の構築に貢献したい。趣味は OSS 開発、筋トレ、観葉植物。

## 職歴

**SmartHR / シニアプラットフォームエンジニア** — 2024年6月〜現在

HR テック領域のプラットフォーム開発、システムアーキテクチャ設計・再設計、パフォーマンス最適化。

**Makuake / シニアソフトウェアエンジニア・リードエンジニア** — 2018年12月〜2024年5月

プラットフォームチームのリードエンジニアとして、事業拡大に伴うスケーラビリティ課題を解決。認証・認可・通知基盤などの基盤システムの設計・開発をリード。

**Innovator Japan / プロダクトマネージャー・エンジニア** — 2017年5月〜2018年9月

求人関連サービスの新規サービス開発（企画・プロトタイプ）、CMS/EC パッケージ開発、受託 Web サイト開発。

**スタートアップ / Web エンジニア** — 2016年5月〜2017年3月

求人サービスの機能開発・UI 実装、アナリティクスを活用したサービス成長。

**フリーランス / Web エンジニア** — 2015年6月〜2016年5月

多様なクライアント・プロジェクトにおける受託 Web 開発。
